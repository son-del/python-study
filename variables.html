파이썬 변수와 자료형

데이터를 저장하고 다루는 가장 기본적인 개념

1. 변수란?

변수는 값을 저장하기 위한 이름표예요. 파이썬에서는 변수 선언 시 자료형을 미리 지정하지 않아도 됩니다.

# 변수 선언 예시
name = "Alice"
age = 25
height = 165.4

print(name)
print(age)
print(height)

출력 결과:

Alice
25
165.4

2. 주요 자료형

파이썬에서 자주 쓰이는 기본 자료형은 아래와 같습니다.

  • int: 정수형 (예: 10, -3)
  • float: 실수형 (예: 3.14, -0.5)
  • str: 문자열형 (예: "hello", '파이썬')
  • bool: 논리형 (예: True, False)
# 자료형 예시
a = 10
b = 3.14
c = "파이썬"
d = True

print(type(a))
print(type(b))
print(type(c))
print(type(d))

출력 결과:

<class 'int'>
<class 'float'>
<class 'str'>
<class 'bool'>

3. 여러 값 다루기

변수끼리의 연산과 형 변환도 자주 사용됩니다.

# 숫자 연산
x = 5
y = 2
print("덧셈:", x + y)
print("나눗셈:", x / y)
print("몫:", x // y)
print("제곱:", x ** y)

# 형 변환
num_str = "100"
num = int(num_str)
print("문자열 → 정수:", num + 50)

4. 여러 변수 한 줄에 선언하기

x, y, z = 1, 2, 3
print(x, y, z)

또는 같은 값을 여러 변수에 동시에 할당할 수도 있습니다.

a = b = c = 10
print(a, b, c)

5. 문자열 포매팅

변수의 값을 문자열 안에 넣는 대표적인 방법입니다.

name = "민수"
age = 20
print(f"안녕하세요, {name}님! 나이는 {age}살입니다.")

출력 결과:

안녕하세요, 민수님! 나이는 20살입니다.
